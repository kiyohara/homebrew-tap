# Homebrew Tap for slapex

This tap provides the Homebrew Cask for [slapex](https://github.com/kiyohara/slapex).

## Install

```sh
brew install --cask kiyohara/tap/slapex
```

## Verify

```sh
slapex --version
```

## Release Maintenance

The `slapex` cask is generated from the main `kiyohara/slapex` repository by GoReleaser.

The initial `v1.0.0` cask was bootstrapped manually because that release existed before the tap automation was added.
